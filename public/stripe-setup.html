💳 Stripe Setup

Configure Stripe to accept payments for MoltBot Cloud subscriptions.

Checking Stripe status...

Step 1: Get Stripe API Key

Create Stripe Account

1. Go to dashboard.stripe.com/register
2. Create a free account (test mode is fine to start)
3. Go to Developers → API Keys
4. Copy your Secret key (starts with sk_test_)

Step 2: Enter Key

Set API Key

This saves the key as an environment variable and creates subscription products in Stripe.

Step 3: Webhook (Optional)

Forward webhooks for real-time subscription events

stripe listen --forward-to http://localhost:3000/api/billing/webhook

This lets MoltBot Cloud handle subscription created/cancelled/failed events in real time.
